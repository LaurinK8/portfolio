Mobile Maps App (Prototype)

Overview

A mobile‑first mapping application prototype with address search, GPS location, and interactive markers, built as part of a university course.

Core Requirements

  • Platform support (Android or iOS)
  • Permission handling for location access
  • Interactive map view with zoom & drag
  • Address search with marker placement
  • GPS‑based current location marker
  • Custom markers for search vs. GPS location
  • Loading indicators during geolocation
  • Persist last map viewport between sessions
